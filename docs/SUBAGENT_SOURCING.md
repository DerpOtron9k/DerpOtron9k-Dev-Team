## Default Upstream Source

Use [VoltAgent awesome-codex-subagents](https://github.com/VoltAgent/awesome-codex-subagents) as the default upstream catalog for subagent TOML files.

This repository keeps that catalog under [Subagent Directory](/R:/GitRepo/DerpOtron9k-Dev-Team/Subagent%20Directory).

## Starter Rule

The root repo includes only the minimal delivery ring needed to start:

- `product-manager`
- `scrum-master`
- `scrum-coach`

Kickoff should install or recommend the rest of the council based on the project's actual needs.
